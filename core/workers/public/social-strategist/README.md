# social-strategist

Content planning and draft creation worker for the social-team.

## Skills

| Skill | What it does |
|-------|-------------|
| `ideate` | Take a raw idea -> produce draft posts |
| `suggest` | Research and suggest what to post next |
| `calendar` | Plan a week of content |
| `adapt` | Adapt a draft for different profile/platform |

## Usage

```bash
/run social-strategist ideate "launched {company} Docs v0.2.2 with fullscreen and collapsed sidebar" --profile {company}
/run social-strategist suggest --profile personal
/run social-strategist calendar --profile {company}
/run social-strategist adapt post-035 --from personal --to {company}
```
