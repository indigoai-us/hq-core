# HQ Cheatsheet

Daily reference card. Pin this somewhere visible.

---

## The Loop

```
/startwork  →  /brainstorm  →  /plan  →  /run-project  →  /handoff
```

---

## Essential Commands

| Command | When to Use |
|---------|-------------|
| `/startwork` | Start of every session — reads your last handoff, shows your board |
| `/brainstorm` | Fuzzy idea — you know the direction but not the shape |
| `/plan` | Concrete project — turns an idea into stories with acceptance criteria |
| `/run-project` | Execute a PRD — runs stories through the Ralph Loop |
| `/handoff` | End of every session — logs work, preserves state |
| `/learn` | After something worked or broke — captures corrections permanently |

## Building Commands

| Command | When to Use |
|---------|-------------|
| `/newworker` | Create a specialized worker for a new domain |
| `/personal-interview` | Build your voice profile and preferences |
| `/checkpoint` | Mid-session save without fully closing |
| `/search` | Find anything in HQ by topic |

## Maintenance Commands

| Command | When to Use |
|---------|-------------|
| `/reanchor` | HQ feels cluttered or slow — garden cleanup |
| `/garden` | Audit and clean stale content |
| `/cleanup` | Deep clean with INDEX rebuild |

---

## Daily Cadence

```
Morning     /startwork [company]       Pick up where you left off
Execute     /plan + /run-project        Ship the work
Capture     /learn                     Teach what you discovered
Close       /handoff                   Save state, end session
```

---

## The Ralph Loop

```
Plan  →  Execute  →  Verify  →  Iterate
         (fresh context per task, back pressure at every step)
```

---

## Rules of Thumb

- **Plan mode always** — every task, no exceptions
- **One thing at a time** — fresh context per task
- **Be bossy but naive** — clear outcomes, not implementation details
- **Fresh sessions > long ones** — `/handoff` early and often
- **Teach, don't just correct** — `/learn` makes the system smarter

---

## When Things Go Wrong

| Problem | Fix |
|---------|-----|
| Session feels slow or confused | `/handoff` and start fresh |
| Worker keeps making the same mistake | `/learn` to capture the correction |
| HQ feels cluttered | `/reanchor` to clean up |
| Don't know what to work on | `/startwork` reads your board |
| Project is too vague to execute | `/brainstorm` first, then `/plan` |
