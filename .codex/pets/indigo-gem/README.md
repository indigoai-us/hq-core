# Indigo Gem

Indigo Gem is a friendly classic diamond-shaped Codex pet with a faceted indigo body, tiny limbs, and a bright expression.

Install by copying this folder to:

```text
~/.codex/pets/indigo-gem
```

The installable files are `pet.json` and `spritesheet.webp`. `contact-sheet.png` is included as a row-by-row animation preview.
